Data Structures & Abstractions

Introduction to PASS ICT283


Reliable software powers the world.

It is our responsibility as programmers to write code that is simple, understandable, efficient, maintainable and reasonably free of bugs.

Modern applications store and compute vast amounts of data and programmers need to understand how to handle data in the most efficient way possible.

The purpose of data structures and algorithms is to give programmers the tools they need to make good programming decisions which in turn lead to well-designed and efficient software.

Programming is all about data structures and algorithms. Data structures are used to hold data while algorithms are used to solve the problem using that data.

Data structures and algorithms goes through solutions to standard problems in detail and gives you an insight into how efficient it is to use each one of them. It also teaches you the science of evaluating the efficiency of an algorithm. This enables you to choose the best of various choices.
